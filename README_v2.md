# SBC6526
A 6502 based SBC, capable of using a MOS6526, or the new replacement 74HCT6526

Being my first attempt at creating a SBC, v1 had many flaws. There were some pull-up resistors missing, the PCB layout was a mess, the bootstrap code was... well... unreadable to be honest. Moreover, the CAD tool I choose for v1, Eagle Cad, has a nasty licensing which is pulling away many users. I decides to make the transition to KiCAD for the v2. It's still a work in progress, and completely untested, but feel free to take a look is you like.

This readme will be updated as soon as new content is commited to the repository.
